# 03 · Physical Security

## 1. Concepto general

La seguridad física protege personas, activos, información y equipos frente a amenazas físicas o ambientales, garantizando la **Confidentiality, Integrity y Availability (CIA)** mediante controles tangibles.

### Objetivos principales

* Evitar accesos no autorizados a instalaciones
* Proteger equipos críticos y datos físicos
* Prevenir daños ambientales o por desastres
* Asegurar la continuidad operativa

---

## 2. Capas de seguridad física

| Capa                   | Ejemplos de controles                                                   | Propósito                                   |
| ---------------------- | ----------------------------------------------------------------------- | ------------------------------------------- |
| **Perímetro exterior** | Vallas, bolardos, iluminación, guardias, cámaras PTZ, señalización      | Detección temprana y disuasión              |
| **Perímetro interior** | Cerraduras electrónicas, torniquetes, vestíbulos, control de visitantes | Controlar y registrar accesos               |
| **Zonas restringidas** | Biometría, tarjetas RFID, alarmas de puerta                             | Limitar acceso a personal autorizado        |
| **Activos críticos**   | Armarios cerrados, racks con candado, jaulas de servidores              | Proteger hardware sensible                  |
| **Entorno ambiental**  | HVAC, detectores de humo, UPS, generadores                              | Condiciones seguras y continuidad eléctrica |

---

## 3. Tipos de controles físicos

| Tipo              | Descripción                             | Ejemplo                               |
| ----------------- | --------------------------------------- | ------------------------------------- |
| **Preventivo**    | Evita intrusión o daño antes de ocurrir | Cerraduras, cercas, guardias          |
| **Detectivo**     | Identifica actividad sospechosa         | Cámaras, sensores, alarmas            |
| **Correctivo**    | Recupera o mitiga tras incidente        | Sistemas antiincendios, UPS           |
| **Disuasorio**    | Reduce la probabilidad de ataque        | Iluminación visible, señales          |
| **Compensatorio** | Sustituye un control faltante           | Guardias si falla sistema electrónico |

---

## 4. Dispositivos y tecnologías comunes

* **Bollards y barreras**: impiden acceso vehicular no autorizado
* **Fencing (vallas)**: delimitación perimetral
* **Lighting (iluminación)**: disuasión y visibilidad
* **Mantrap / Access Control Vestibule**: zona de verificación entre dos puertas
* **Badges y RFID**: autenticación física
* **Biometría**: huella, iris, reconocimiento facial
* **CCTV**: monitoreo y evidencia
* **Guardias de seguridad**: control y respuesta
* **Sensores**:

  * Infrarrojos (IR) → detección por calor
  * Ultrasónicos → detección por sonido
  * Presión / microondas → vibración o apertura
* **Puertas reforzadas y cerraduras electromagnéticas**

---

## 5. Amenazas físicas y ambientales

| Tipo                    | Ejemplo                             | Medida preventiva                      |
| ----------------------- | ----------------------------------- | -------------------------------------- |
| **Intrusión física**    | Tailgating, clonación de tarjetas   | Mantrap, guardias, políticas estrictas |
| **Desastres naturales** | Inundaciones, incendios, terremotos | DRP, ubicación elevada de equipos      |
| **Daños ambientales**   | Temperatura, humedad, polvo         | HVAC, sensores, mantenimiento          |
| **Robo / sabotaje**     | Sustitución de hardware, robos      | CCTV, inventario, cableado seguro      |
| **Ataques internos**    | Empleados maliciosos                | Supervisión, control de privilegios    |

---

## 6. Mejores prácticas de gestión

* Registro y auditoría de accesos físicos
* Aplicar principio de **menor acceso necesario**
* Revocar accesos de personal inactivo o despedido
* Políticas estrictas para visitantes
* Integración seguridad física + cibernética (SIEM + CCTV)
* Pruebas periódicas de alarmas y sistemas eléctricos
* Documentación de planos, accesos y protocolos de emergencia
